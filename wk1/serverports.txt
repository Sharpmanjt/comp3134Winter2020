1. What is the definition of server port?
This is the port on the server that an outside source can connect to in order to transmit data. The server has a port 
for the different services available. 

2. What is server port range?
0 - 65535

3. How many bits does number above represent?
16 bits

4. State 10 common server ports
80 - http
3306 - mySQL
25 - SMPT
22 - SSH
20 - FTP
53 - DNS
28 - Telnet
220 - IMAP
311 - MaxOS X Server Admin
5000 - heroku console
